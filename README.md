## Run Locally

Clone the project

```bash
  git clone https://github.com/nguyen-duc-loc/Registry-Total.git
```

Go to the project directory

```bash
  cd Registry-Total/client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
